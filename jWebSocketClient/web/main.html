jWebSocket.org Client

This web site demostrates the capabilities of the jWebSocket package. You can run this site either in an arbitrary folder on your local file system and access a stand-alone version of the jWebSocket Server jWebSocketServer.jar or you can run it within the web application jWebSocketAppServer.war package by simply deploying the .war-file in Tomcat. For Browsers which do not yet natively support WebSockets please refer to the required server side provisions described at http://jWebSocket.org.


To integrate the jWebSocket client into your web application please embed the jWebSocket.js file into your environment. In the Chat and other demos provided with jWebSocket you will find some easy examples of how to use the jWebSocket API.
